# boiler-monitoring-web

Sistem monitoring suhu boiler industri berbasis Rust yang digunakan untuk memonitor suhu boiler secara realtime, mendeteksi kondisi overheat, serta melakukan stabilisasi suhu otomatis.

---

## Fitur Program

- Monitoring suhu boiler realtime
- Alarm overheat otomatis
- Pendinginan otomatis
- Kontrol suhu boiler

---

## Cara Menjalankan Program

### 1. Clone repository

```bash
git clone https://github.com/Athaliamps13/boiler-monitoring-web.git
```

### 2. Masuk ke folder project

```bash
cd boiler-monitoring-web
```

### 3. Jalankan program Rust

```bash
cargo run
```
